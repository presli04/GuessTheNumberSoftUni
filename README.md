# GuessTheNumberSoftUni
This is a simple guess the number game from my SoftUni course.
## Description
This is a guess the number game written in python.
The computer randomly selects a number beetwen 1 and 100 and the player has to guess it in a limited number of tries.
## Game rules
1.The computer generates a random number between 1 and 100.
2.The player selects a difficulty:
  easy-5 attempts
  medium-3 attempts
  hard-1 attemps
3. The game ends when the player guess the number or when he gives up.
4. Points are awarded based on the difficulty and the number of tries it took.
